android.ichat
================

### 仿微信的一个聊天APP
使用Mina作为客户端的TCP通信框架，通信协议的载体使用的Protocol Buffer，使用Volley作为HTTP框架，数据库方面采用的greenDAO。<br />
程序中主要覆盖的功能有：登录、注册、个人信息维护，文字、表情、语音的接收与发送。<br /><br />
